# HR Data Analysis Dashboard

## Project Overview
This project is an HR Analytics Dashboard created using Power BI to analyze employee data and generate business insights.

## Tools Used
- Power BI
- Power Query
- DAX
- Excel

## Dashboard Features
- Total Employees
- Average Age
- Average Monthly Income
- Salary by Department
- Gender by Department
- Gender by Job Role
- Business Travel Analysis
- Work-Life Balance Analysis
- Education Field Analysis
- Marital Status by Job Role

## Key Insights
- Analyzed employee demographics and salary distribution.
- Compared gender across departments and job roles.
- Studied business travel and monthly income trends.
- Built interactive filters for better data exploration.
